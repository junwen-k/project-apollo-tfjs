This repository contains the source code that are used to train the machine learning model powered by [TensorflowJS](https://www.tensorflow.org/js). The dataset are provided by RHB bank, with the consent of using it solely for research purposes. ## Installation

1. Install dependencies with `npm` or `yarn`.

        npm i
        yarnpkg

2. Build source codes.

        npm run build
        yarn build

3. Train model.

        npm run start
        yarn start

The trained model can be found under `dist/models/${MODEL_NAME}`.
